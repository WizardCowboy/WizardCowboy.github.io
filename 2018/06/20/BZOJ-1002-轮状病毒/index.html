[BZOJ-1002]轮状病毒

DP高精度
  • 递推DP
  • 高精度

Description


如上图n轮状基由n个不同的基原子和圆心处的核原子构成。

n轮状病毒的产生就是n轮状基中删去若干条边，使得各原子间有唯一的信息通道。
求有多少个不同的n轮状病毒。

Solution

这是一道[FJOI2007]的题。
打表找规律，我们发现F[i] = 3 * F[i - 1] - F[i - 2] + 2;

Notice

我们发现可能会超出long long，所以我们要用到高精度。
